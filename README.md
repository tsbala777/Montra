# Montra - Financial Tracking App

Montra is a comprehensive financial tracking application built with React, TypeScript, and Vite. It helps users manage their income, expenses, budgets, and savings goals with a premium, user-friendly interface.

## Features
- **Dashboard**: Overview of finances with charts and key stats.
- **Transactions**: Track income and expenses.
- **Budgets**: Set category-wise spending limits.
- **Goals**: Create and track savings goals.
- **Investments**: Monitor investment portfolio.
- **Analytics**: Detailed financial insights.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
